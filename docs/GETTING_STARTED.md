## FAQ

### Is Satan6x6 safe?
Satan6x6 operates on Solana mainnet with on-chain verifiable transactions. Core launch pipeline uses industry-standard Metaplex and Meteora SDKs. The bot is built on [OpenClaw](https://github.com/openclaw) — a battle-tested open-source agent framework. Custom integration code is private during development for competitive reasons.

### Why Metaplex + Meteora?
- **Metaplex** — Solana's standard for NFT/token metadata
- **Meteora DAMM v2** — permissionless pools with flexible fee tiers, single-sided liquidity option

### Why OpenClaw?
- **Local-first** — data stays on our infrastructure, not third-party servers
- **MIT-licensed** — no vendor lock-in, full transparency
- **Proven architecture** — battle-tested three-layer design (channel/brain/body)
- **Messaging-native** — built for Telegram/Discord/WhatsApp from day one

### Can I run my own instance?
The core launch pipeline is currently closed-source. Infrastructure components (skills, utilities) are available in `/examples/`. Full open source decision pending Phase 7.

### Is this financial advice?
**No.** Everything is experimental software. DYOR before any token purchase or launch decision.
